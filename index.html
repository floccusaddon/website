Sync backends

Floccus allows you to sync your bookmarks via a variety of services. These are the different backend options.

Nextcloud Bookmarks

Nextcloud is a content collaboration platform which can be selfhosted and also offers a Bookmarks app installable from its app store. With the Nextcloud Bookmarks app you can access your bookmarks via a beautiful web UI and sync them with your browser via floccus.

More about Bookmarks

Google Drive

If you have a Google account, you can easily sync your bookmarks via an encrypted file in your Google Drive. Google will be none the wiser and you can sync your bookmarks across all browsers.

WebDAV

If you have a WebDAV file server at hand, like any version of nextcloud, owncloud, box.com or with any WebDAV server, commercially or self-hosted, you can sync your using floccus.

Local file and more

You can also simply sync a local file, using the LoFloccus desktop app allowing you to sync that file to other computers using your file syncing solution, like Dropbox, Syncthing, rsync, etc.
